My experience at working for website agency has thought me a lot. I know that having good coding skills is important but not enough to build quality website. I believe there are a few key areas which have to be taken into considreration in order to achieve the best user experience.

For instance, I have learned how important it is to develop for cross-browser compatibility and for any device's size. Below are the rules I believe should be taken into account when building a website.

Speed

Fast load times and lag free interaction.

Responsiveness

Websites which work on any device, big or small.

Accessibility

Web sites which can be used by as many people as possible.

Dynamic

Have some fun and bring pages to live.

Intuitive UX/UI

Lets try and make the pages as easy to use as possible
