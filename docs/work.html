Skillset

I have knowledge and experience with the following:

  • Front end technologies: HTML, CSS, JS
  • Front end libraries: Bootstrap, SASS, jQuery, familiar with React
  • Back end: CMS's like WordPress, Umbraco and Kentico
  • Back end: apps like Woocommerce
  • Back end: small exposure to PHP and Laravel
  • Basic experience with Webpack, Gulp, npm an JSON
  • Source control: Git with GitHub and Git clients (GitHub Desktop and Tortoise Git)
  • Tools: VS Code, VS, Atom, Docker and Azure DevOps

My experience at working for website agency thought me a lot about web development. These are some of the important principles I have learned.

Speed

Fast load times and lag free interaction.

Responsiveness

Websites which work on any device, big or small.

Accessibility

Websites which can be accessible to as many people as possible.

Dynamic

Have some fun and bring pages to live.

Intuitive UX/UI

Make the pages as easy to use as possible
